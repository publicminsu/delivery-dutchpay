개요
-------------
학교에서 한 학기 동안 팀 프로젝트로 하였던 썸띵 (배달 더치페이)이다.

기본 아이디어
-------------
기숙사 거주생이 배달 음식을 시킬 때 최소 주문 금액을 채우기 힘들다는 점에서 나온 아이디어.
근처 인원과 같은 음식점에서 음식을 주문하기에 1건의 배달 팁을 나눠서 낼 수 있단 점에서 좋은 아이디어였다.
